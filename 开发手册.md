### 项目架构

本项目分为两大核心模块：`back_test`（回测系统）和 `live`（实盘交易系统），结构清晰，便于独立开发和维护。

```
red_trade/
├── back_test/                # 回测模块
│   ├── data/                 # 存放历史数据 (被 .gitignore 忽略)
│   ├── results/              # 存放回测结果 (被 .gitignore 忽略)
│   ├── src/                  # 回测核心源码
│   │   ├── acquisition.py    # 数据获取与处理
│   │   ├── strategy.py       # 交易策略定义
│   │   ├── processing.py     # 回测数据处理
│   │   └── utils.py          # 公共工具函数
│   └── bt_main.py            # 回测主入口文件
│
├── live/                     # 实盘模块
│   ├── logs/                 # 存放实盘日志 (被 .gitignore 忽略)
│   ├── src/                  # 实盘核心源码
│   │   ├── signals.py        # 交易信号生成
│   │   ├── strategy.py       # 实盘交易策略
│   │   ├── exit_mechanism.py # 离场/止损机制
│   │   └── utils.py          # 公共工具函数
│   └── live_main.py          # 实盘主入口文件
│
├── .env                      # 环境变量 (API密钥等)
├── .gitignore                # Git 忽略文件配置
├── requirements.txt          # Python 依赖包
├── 开发手册.md                # 本开发手册
└── README.md                 # 项目说明
```

- **`back_test` 模块**:
  - **`bt_main.py`**: 运行回测的起点。它负责加载数据、配置策略参数、执行回测并生成结果。

- **`live` 模块**:
  - **`live_main.py`**: 启动实盘交易程序的入口。
